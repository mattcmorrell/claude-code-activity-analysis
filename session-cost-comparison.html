Fixed Assumptions

Tasks in session: 1 (clear context between tasks)
Base context: 25K tokens (system prompt, tools, CLAUDE.md)
Per turn (text): +5K tokens (productive) / +6K (fix-up)
Output per turn: 50% of new input tokens that turn (~2,500 typical text, ~3,650 with screenshot). Changing this doesn't shift the Opus/Sonnet ratio — all pricing tiers are proportional between models.
Screenshots: 1 per turn, 4K crop — ~2,300 tokens (4.6/Sonnet/Haiku) / ~3,733 tokens (4.7, higher res cap)
Haiku penalty: 3× more turns than Sonnet (weaker model, more retries)
Opus 4.7 tokenizer: +30% token inflation on text and output (not images)
Cache model: Turn 1 = cold start (all cache writes). Turn 2+ = cache reads + 2% miss rate
